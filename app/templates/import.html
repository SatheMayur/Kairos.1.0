📂 CSV Upload (Naukri / WorkIndia)
📋 How to Import from Naukri

Naukri is your primary source. Follow these steps to import applicants:

  1. Log in to naukri.com → Employer Dashboard
  2. Go to Manage Jobs → click your active job
  3. Click View Applications
  4. Click Download / Export CSV
  5. Upload that CSV file here using Import & Score ←

✓ What happens automatically

System scores every applicant, shortlists top candidates, and sends WhatsApp + email outreach — all in one click.

🔗 Smart URL Import (CAD Crowd / Portfolio)

Paste any public profile URL — CAD Crowd, portfolio sites, WorkIndia, Apna, Shine. Claude reads the page and extracts candidate data automatically.

🔗 Bulk URL Import (up to 20 profiles)

Paste one URL per line. All profiles are fetched and scored in parallel.

✍️ Manual Entry (paste from portal)

Enter candidate details one by one. Useful for WorkIndia handpicked candidates or LinkedIn profiles.
