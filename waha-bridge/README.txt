K. Girdharlal International — WhatsApp Bridge
==============================================

HOW TO START:
  Double-click: start-whatsapp.bat
  Keep the window open while working.

FIRST TIME SETUP (auto-start on boot):
  Double-click: setup-autostart.bat
  After this, WhatsApp starts every time your computer turns on.

WHAT EACH FILE DOES:
  bridge.js           — The main WhatsApp program (do not edit)
  start-whatsapp.bat  — Start WhatsApp manually (double-click this)
  setup-autostart.bat — Set up auto-start on Windows boot (run once)
  session/            — Your WhatsApp login is saved here (do not delete)

IF WHATSAPP DISCONNECTS:
  1. Double-click start-whatsapp.bat
  2. Go to your recruitment system dashboard
  3. Open WhatsApp page and scan the QR code

TO SWITCH TO A DIFFERENT WHATSAPP NUMBER:
  No need to delete any files. While this window is running:
  1. Open the WhatsApp page in your dashboard
  2. Click the "Switch Number" button
  3. A new QR code appears — scan it with the NEW phone
  (The old number is logged out automatically.)

NEED HELP?
  Contact your system administrator.
